数组和指针的区别

概念

数组：用于储存多个相同类型数据的集合。数组名是首元素的地址；
指针：变量，存放的是其他变量在内存中的地址。指针名指向内存的首地址。

区别

1.赋值。同类型指针变量可以相互赋值，但数组不行，只能一个一个元素赋值或copy。
2.存储方式。
数组在内存中连续存放，开辟一块连续的内存空间。数组根据数组下标访问，数组的存储空间，在静态区或栈。
指针本身是变量，存储空间不能确定。
3.求sizeof
数组所占存储空间的大小：sizeof(数组名)，或sizeof(数据类型)
32位平台下，无论指针的类型是什么，sizeof(指针名)都是4；64位平台，sizeof(指针名)=8。
4.初始化 

//数组
int a[5] = {0};
char b[] = "hello";
char c[] = {'h', 'e', 'l', 'l', 'o', '\0'};
int* arr = new int[10];//动态创建一维数组
//指针
int* p = new int(0);
delete p;//指向对象

int* p1 = new int[10];//指向数组
delete[] p1;

string* p2 = new string;//指向类
delete p2;

int** pp = &p;//指向指针的指针
**pp = 10;

5.指针操作
数组名的指针操作： 

int a[3][4];
int (*p)[4];//定义一个数组指针，指向含4个元素的一维数组
p = a;//把二维数组的首地址赋给p，a[0]或a[0][0]
p++;//该语句执行后，p指向a的下一行a[1][].
//数组指针也称为指向一维数组的指针，行指针。
//访问数组中第i行第j列的元素，有几种操作方式：
*(p[i] + j)
*(*(p + i) + j)
(*(p + i))[j]
p[i][j]
//优先级：() > [] > *

6.指针的数据操作 

char* str = "hello";
str[2] = 'a';
*(str + 2) = 'b';
///两种操作都合法

什么是函数指针，如何定义，有哪些使用场景

含义

函数指针是指向函数的指针变量。函数有入口地址，函数指针指向这个入口地址。

定义

int func(int a){
    return 0;
}
int (*f)(int a);
f = &func;

应用场景

1.回调。callback，调用别人提供的API（应用程序编程接口），称为call；别人的库调用我们的函数称为callback。

静态变量什么时候初始化

c语言的全局和静态变量，初始化发生在任何代码执行之前，属于编译期初始化；
c++规定，全局或静态对象当且仅当对象首次使用时采进行构造。

nullptr调用成员函数可以吗？原因

可以。编译时对象就绑定了函数地址，和指针空不空无关。

什么是野指针，怎么产生的，如何避免

野指针是指针指向的位置不可知（随机、不正确、无明确限制的）；
产生原因：释放内存后不及时清空，依然指向了该内存，可能出现非法访问的错误。
避免办法：初始化置NULL、申请内存后判空、指针释放后置NULL、使用智能指针。 

int *p = NULL;//初始化
p = new int[10];
assert(p != NULL);//判空
free(p);
p = NULL;//释放后置NULL
